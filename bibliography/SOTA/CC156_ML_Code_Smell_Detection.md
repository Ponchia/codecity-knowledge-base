# CC156: Machine Learning-Based Methods for Code Smell Detection: A Survey

**Source URL**: https://www.mdpi.com/2076-3417/14/14/6149
**Journal**: Applied Sciences 2024, 14(14), 6149
**DOI**: https://doi.org/10.3390/app14146149
**Gap Category**: AI/ML Code Analysis

## Overview

This comprehensive survey provides a systematic review of machine learning approaches for code smell detection from 2005 to 2024, covering 42 carefully selected research papers.

## Key Contributions

### Scope
- Over 30 code smells discussed
- Survey covers ML algorithms from 2005 to 2024
- 42 research papers selected based on quality
- Focus on research from journals and conferences where ML algorithms were applied

### ML Techniques Covered
- Decision Tree
- Bayesian learning
- Ensemble learning (Random Forest, Gradient Boosting Tree)
- SVM (Support Vector Machine)
- Rule-Based Learning
- Neural networks (Multilayer Perceptron, Deep Learning)
- Genetic Algorithm (GA)
- Naive Bayes
- GA CFS (Correlation-based Feature Selection)

### Key Findings
- 17 research articles from IEEE indicating its prominence in this field
- Evaluated efficacy of ML algorithms on open-source and project datasets
- Identified threats and obstacles:
  - Lack of standardized code smell definitions
  - Difficulty of feature selection
  - Challenges of handling large-scale data

## Publication Sources
Predominantly IEEE publications, indicating the field's prominence in software engineering research.
