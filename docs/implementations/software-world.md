# Software World

## Overview

Early city-metaphor implementation using the Maverik VR toolkit. CC015 describes Java classes mapped to districts, with methods represented as buildings.

## CC015 Table 1

- **Language**: Java
- **VR**: Maverik
- **Building**: function
- **Source code**: n/a
- **Static**: LOC, #methods, public/private, #parameters, parameter types
- **Dynamic**: n/a
- **Instrumentation**: n/a

## Notable Details (from CC015)

- Method building height uses a LOC-derived scale (“one storey per 10 LOC”).
- Parameters are depicted as exterior doors.
- Lighter/darker colors indicate public vs private.
- CC015 notes this fine granularity improves detail but reduces scalability.
