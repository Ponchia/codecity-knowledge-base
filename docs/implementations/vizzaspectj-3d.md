# VizzAspectJ-3D

## Overview

Eclipse plugin that extends CodeCity-style visualization to AspectJ, representing both classes and aspects as buildings. CC015 reports aspect height based on number of pointcuts and advice, with different color-coded views for complexity and dependencies.
