# UML-City

## Overview

City-like visualization for UML diagrams by combining MetaView (relationships across reduced-size UML views) and MetricView (metrics overlaid as height/color/shape). CC015 reports a user study showing improved speed and correctness versus conventional 2D UML diagrams.
